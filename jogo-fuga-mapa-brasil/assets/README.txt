Coloque aqui as imagens PNG reais das cabeças usadas como obstáculos.

Arquivos esperados (nomes exatos):
- lula.png
- bolsonaro.png

Requisitos:
- Formato: PNG
- Tamanho recomendado: ~128x128 a 256x256 (será redimensionado para 56x56)
- Fundo: preferencialmente com transparência (fundo recortado) para melhor visual.

Como usar:
1) Salve as imagens acima nesta pasta (assets).
2) Abra o arquivo "../index.html" no navegador. Se os arquivos existirem, o jogo os usará automaticamente.
3) Caso as imagens não estejam presentes ou falhem ao carregar, o jogo usa um desenho simples como fallback.
